# Pulgarcito el cuento

Proyecto web introductorio desarrollado con HTML y CSS como parte de una práctica de aprendizaje en el curso de Digitech Cáritas. La idea principal fue crear una página sencilla, organizada y visualmente atractiva, mientras se practicaban conceptos básicos de estructura web y estilos.

## Descripción del proyecto

Esta página presenta una adaptación del cuento de "Pulgarcito", combinando contenido narrativo con diseño básico. El proyecto sirve como ejemplo de cómo construir una web estática con:

- Estructura semántica en HTML
- Estilización con CSS
- Imágenes y contenido visual
- Navegación entre páginas

## Demo

El proyecto está disponible en GitHub Pages:

## Características

- Página principal con contenido narrativo
- Sección adicional sobre el proyecto
- Diseño simple y limpio
- Uso de estilos CSS para mejorar la presentación
- Estructura adaptable para su visualización en navegador

## Estructura del proyecto

- index.html: página principal
- aboutMe.html: página adicional de información
- style.css: estilos generales
- index.css: estilos complementarios
- imgPulgarcito.png: imagen principal del proyecto

## Cómo ejecutar el proyecto

Puedes ver el proyecto abriendo el archivo index.html en tu navegador.

Si prefieres trabajar con un servidor local, puedes usar:

- Live Server en Visual Studio Code
- XAMPP o un servidor local similar
